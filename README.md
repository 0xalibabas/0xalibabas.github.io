## Local Development

```bash
git clone https://github.com/0xalibabas/0xalibabas.github.io.git
cd 0xalibabas.github.io
git submodule update --init --recursive
hugo server --buildDrafts
```

## Writing Posts

```bash
hugo new content posts/your-post-title.md
```
